# 🐾 DOGMA - 2D Pixel Art Platformer

## Overview
**DOGMA** is a high-energy 2D platformer where players take on the role of **Fenrir**, a heroic dog on a mission to rescue his kidnapped owner. This project demonstrates my ability to build a cohesive game experience, from character mechanics to UI implementation.

## 🛠️ Core Features & Development Focus

* **Fluid Character Controller:** Developed a responsive 2D movement system for the protagonist (Fenrir), focusing on "game feel" and tight platforming controls.
* **Pixel Art Integration:** Seamlessly integrated custom 2D pixel art assets, maintaining a consistent visual style throughout the game.
* **Game Loop Implementation:** Built a complete experience including a start menu, gameplay levels, and win/loss conditions.
* **UI & UX Design:** Designed and implemented an intuitive user interface that complements the game’s aesthetic and provides clear feedback to the player.

## 💻 Technical Specifications

* **Genre:** 2D Platformer
* **Platform:** PC (Windows Executable)
* **Format:** `.exe`
* **Core Tools:** Asset management and gameplay logic implementation.

## 🚀 Key Learning Outcomes
This project served as a deep dive into the full game development lifecycle—taking a concept from a research stage to a fully packaged, playable build ready for user testing.
